## 📊 一、项目概述

### 1.1 项目背景
Agent Company 是一个基于 OpenClaw 的 AI 团队管理平台，当前版本为网页应用。为提升用户体验和部署便利性，需要转换为桌面应用（exe）并修复已知问题。

### 1.2 产品定位
- **目标用户**: AI 团队管理者、开发者
- **核心价值**: 一站式 AI 团队管理、任务分配、财务统计
- **使用场景**: 桌面办公、本地部署、离线使用

### 1.3 当前状态
| 模块 | 状态 | 技术栈 |
|------|------|--------|
| 前端 | ✅ 运行中 | React + Ant Design + Vite |
| Agent API | ✅ 运行中 | Node.js + Express + SQLite |
| 论坛 API | ✅ 运行中 | Node.js + Express + SQLite |
| 监控 API | ⚠️ 未集成 | Python Flask |
| Bridge 服务 | ⚠️ 依赖 Python | Python |

## 🎯 二、核心需求

### 2.1 桌面应用转换（P0）

#### 需求描述
将网页应用转换为 Windows 桌面应用（.exe），支持双击启动、系统托盘、原生通知。

#### 技术方案
- **框架**: Electron（推荐）或 Tauri
- **打包工具**: electron-builder
- **更新机制**: electron-updater

#### 功能清单
- [ ] 双击 exe 启动应用
- [ ] 系统托盘图标
- [ ] 最小化到托盘
- [ ] 原生系统通知
- [ ] 全局快捷键（可选）
- [ ] 开机自启动（可选）
- [ ] 自动更新

#### 验收标准
1. 双击 exe 能在 5 秒内启动应用
2. 最小化后在系统托盘显示图标
3. 右键托盘菜单包含"打开"、"退出"
4. 应用关闭后不残留进程

### 2.2 Bug 修复（P0）

#### 高优先级 Bug 清单

| ID | 问题 | 优先级 | 预计工时 | 状态 |
|----|------|--------|----------|------|
| BUG-001 | Bridge 服务依赖 Python | P0 | 4h | 🔵 待办 |
| BUG-002 | 硬编码 localhost | P0 | 2h | 🔵 待办 |
| BUG-003 | WebSocket 无重连机制 | P0 | 3h | 🔵 待办 |
| BUG-004 | 无错误边界处理 | P0 | 2h | 🔵 待办 |
| BUG-005 | JWT Secret 使用默认值 | P0 | 1h | 🔵 待办 |
| BUG-006 | 数据库连接未关闭 | P0 | 2h | 🔵 待办 |

#### 详细需求

**BUG-001: Bridge 服务 Node.js 重写**
- 当前: Python 轮询通知
- 目标: 用 Node.js 实现相同功能
- 验收: 通知功能正常，无需 Python 环境

**BUG-002: 硬编码 localhost**
- 当前: 多处硬编码 `localhost:3000`
- 目标: 使用环境变量 `API_HOST` 和 `API_PORT`
- 验收: 支持配置主机名和端口

**BUG-003: WebSocket 无重连**
- 当前: 断开后永久离线
- 目标: 自动重连（指数退避）
- 验收: 网络恢复后 30 秒内重连成功

**BUG-004: 无错误边界**
- 当前: 单组件错误导致白屏
- 目标: React Error Boundary
- 验收: 局部错误不影响其他功能

**BUG-005: JWT Secret 默认值**
- 当前: `.env.example` 使用示例密钥
- 目标: 首次启动生成随机密钥
- 验收: 生产环境使用唯一密钥

**BUG-006: 数据库连接未关闭**
- 当前: 进程退出时连接泄漏
- 目标: 监听 SIGTERM/SIGINT 优雅关闭
- 验收: 正常退出无资源泄漏

## 🏗️ 三、技术架构

### 3.1 整体架构

```
┌─────────────────────────────────────────────────────┐
│                  Electron 主进程                     │
│  (main.js - Node.js)                                │
│  - 窗口管理                                          │
│  - 系统托盘                                          │
│  - 原生通知                                          │
│  - 自动更新                                          │
└─────────────────────────────────────────────────────┘
                          │
                          │ 加载
                          ▼
┌─────────────────────────────────────────────────────┐
│               Electron 渲染进程                      │
│  (React 前端 - 端口 3002)                            │
│  - 仪表盘                                            │
│  - 任务管理                                          │
│  - 论坛系统                                          │
│  - Agent 管理                                        │
└─────────────────────────────────────────────────────┘
                          │
                          │ API 调用
                          ▼
┌─────────────────────────────────────────────────────┐
│                  后端服务                            │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐ │
│  │ Agent API   │  │ 论坛 API    │  │ 监控 API    │ │
│  │ 端口 3001   │  │ 端口 3000   │  │ 端口 5000   │ │
│  │ Node.js     │  │ Node.js     │  │ Python      │ │
│  └─────────────┘  └─────────────┘  └─────────────┘ │
└─────────────────────────────────────────────────────┘
                          │
                          │ 数据持久化
                          ▼
┌─────────────────────────────────────────────────────┐
│                  数据库                              │
│  - SQLite (开发环境)                                 │
│  - PostgreSQL (生产环境 - 可选)                      │
└─────────────────────────────────────────────────────┘
```
