## 🎯 P0 场景（MVP 核心）

### 场景 1: 邮件附件自动保存

**用户**: 王强（自由职业设计师）

**场景描述**: Gmail 附件自动保存到 Google Drive，按项目分类

**触发条件**: Gmail 收到带附件的邮件

**执行流程**:
```
1. 检测新邮件带附件
2. 提取附件和发件人信息
3. 按发件人/日期生成文件名
4. 保存到 Google Drive 对应文件夹
5. 标记邮件为"已处理"
6. 发送通知给用户
```

**用户价值**:
- ⏱️ 节省时间：2 小时/周 → 5 分钟/周
- 📁 文件不丢失：自动归档，易查找
- 🧠 减少脑力：无需记住手动处理

**使用频率**: 每天 10-20 次

**复杂度**: 低

**MVP 优先级**: ✅ P0

---

### 场景 2: 文件备份和同步

**用户**: 所有个人用户

**场景描述**: 重要文件自动备份到多个位置（本地 + 云端）

**触发条件**: 文件保存到指定文件夹

**执行流程**:
```
1. 检测文件变化
2. 复制到 Google Drive
3. 复制到 Dropbox
4. 复制到本地 NAS（可选）
5. 记录备份日志
6. 失败时通知用户
```

**用户价值**:
- 🔒 数据安全：三重备份不丢失
- ⏱️ 无需手动：自动完成
- 💰 成本低：利用现有云服务

**使用频率**: 每天多次

**复杂度**: 低

**MVP 优先级**: ✅ P0

### 场景 3: 销售线索自动跟进

**用户**: 张明（创业团队 CTO）

**场景描述**: 网站表单线索自动录入 CRM，发送欢迎邮件，分配销售

**触发条件**: 网站表单提交

**执行流程**:
```
1. 获取表单数据

**用户价值**:
- 📈 转化率提升：线索不遗漏
- ⏱️ 响应及时：秒级跟进
- 👥 协作高效：自动分配

**使用频率**: 每天 10-50 次

**复杂度**: 中

**MVP 优先级**: ✅ P0

### 场景 5: 社交媒体自动发布

**用户**: 李娜（营销机构运营总监）

**场景描述**: 博客文章自动发布到多个社交平台

**触发条件**: 博客发布（RSS 或 webhook）

**执行流程**:
```
1. 获取文章标题、内容、图片
2. 适配各平台格式（字数、标签）

**用户价值**:
- ⏱️ 节省时间：30 分钟/篇 → 1 分钟/篇
- 📈 扩大影响：多平台曝光
- 🎯 格式优化：自动适配

**使用频率**: 每周 2-5 次

**复杂度**: 中

**MVP 优先级**: ✅ P0

## 💡 场景设计原则

1. **高价值**: 节省时间>30 分钟/次
2. **高频次**: 每天/每周使用
3. **低复杂度**: MVP 优先简单场景
4. **可复制**: 多个用户有相同需求
5. **易演示**: 能清晰展示价值
