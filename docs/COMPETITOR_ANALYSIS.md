## 🎯 一、直接竞品分析

### 1. Zapier

**核心功能**: 5000+ 应用集成，可视化工作流构建  
**定价**: $19.99/月起  
**优点**: 集成多、稳定可靠、品牌知名度高  
**缺点**: 价格高、配置繁琐、无 AI 辅助  
**市场份额**: 1000 万 + 用户，$100M+ 年收入

---

### 2. Make (原 Integromat)

**核心功能**: 1000+ 应用集成，可视化场景构建器  
**定价**: $9/月起  
**优点**: 价格比 Zapier 低、可视化编辑器强大  
**缺点**: 学习曲线陡峭、界面复杂  
**市场份额**: 300 万 + 用户

---

### 3. n8n

**核心功能**: 200+ 原生集成，支持自托管  
**定价**: 免费自托管 / €20/月云版  
**优点**: 开源、数据隐私好、自定义能力强  
**缺点**: 需要技术背景、集成数量少  
**市场份额**: 50 万 + 用户，GitHub 35K+ Stars

---

### 4. Microsoft Power Automate

**核心功能**: 300+ 连接器，与 Office 深度集成  
**定价**: $15/用户/月起  
**优点**: 与 Microsoft 365 深度集成、企业级安全  
**缺点**: 非微软生态集成弱、界面复杂  
**市场份额**: 500 万 + 用户（依托 M365）

---

### 5. IFTTT

**核心功能**: 1000+ 应用集成，简单触发器 + 动作  
**定价**: $3.99/月起  
**优点**: 价格低、简单易用、家庭自动化强  
**缺点**: 功能简单、无复杂工作流  
**市场份额**: 2000 万 + 用户

## 🤖 二、AI 竞品分析（v2.0 补充）

### 1. AutoGen (Microsoft)

**公司背景**: 微软研究院，2023 年发布  
**核心功能**: 多 Agent 协作框架，代码执行，对话式任务  
**定价**: 开源免费（仅 API 成本）  
**优点**: 灵活强大、微软背书、社区活跃  
**缺点**: 需要编程、无可视化界面、部署复杂  
**威胁等级**: ⭐⭐⭐⭐（技术用户市场）  
**我们的差异化**: 无需编程、可视化界面、开箱即用

1. **AI 原生**: 自然语言配置工作流（"说人话，自动化"）
